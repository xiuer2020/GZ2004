Linux/CMD常用命令
	目录操作
		cd 进入目录
			./当前目录
			../上级目录
			CMD切换盘符D：
		ls/dir 列出文件
			ls -a:显示隐藏文件
			ls -l:列表方式显示信息
			ls -h:显示文件大小
		pwd 查看当前路径
		mkdir 创建文件夹
		mv 移动/重命名
	文件操作
		touch 新建文件
			新建文件
		vi/vim 编辑文件
			i 插入模式
			esc 退出插入模式
			:wq 保存并退出
		cat 查看文件内容
		stat 查看文件信息
		rm/del 删除文件
			-r
			-h
	其他
		clear/cls 清空终端
		--help 打开帮助
